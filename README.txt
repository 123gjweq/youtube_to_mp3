This script uses requests and the mp3-convert API to save the audio component of youtube videos.
If this script isn't working, you probably have to install requests.
You can do this with pip install requests or pip3 install requests.
This code isn't licensed.

***Make sure you enter the youtube video ID properly.***
***For example, the ID of this link https://www.youtube.com/watch?v=dQw4w9WgXcQ is dQw4w9WgXcQ***
***The ID should only be a few characters long and shouldn't contain {}*&%$./()[]?***
